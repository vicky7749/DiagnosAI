# DiagnosAI - AI Disease Prediction System

A comprehensive web-based disease prediction system leveraging Machine Learning and Deep Learning for accurate diagnosis.

## Features

- **Multi-Disease Prediction**: Supports 8 different diseases
- **High Accuracy**: Average prediction accuracy above 90%
- **User Management**: Secure registration and login system
- **Report Generation**: Save and view diagnostic reports
- **RESTful API**: JSON API for predictions
- **Responsive Design**: Mobile-friendly interface

## Supported Diseases

1. Alzheimer's Disease
2. Brain Tumor
3. COVID-19
4. Diabetes
5. Hepatitis C
6. Kidney Disorders
7. Pneumonia
8. Breast Cancer
