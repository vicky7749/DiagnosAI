About DiagnosAI

Project Overview

DiagnosAI is a comprehensive web-based disease prediction system leveraging Machine Learning (ML) and Deep Learning (DL) to facilitate early and accurate diagnosis. The system supports prediction for eight distinct diseases with an average prediction accuracy above 90%.
